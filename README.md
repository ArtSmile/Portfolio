# Portfolio

This is a personal portfolio website showcasing various case studies and projects. The project features a password-protected entry and responsive design.

## Features
- **Responsive Design**: Adapts seamlessly to mobile and desktop screens.
- **Security**: Includes a simple password lock screen mechanism.
- **Case Studies**: Detailed pages for individual projects (Case 1, Case 2, Case 3).

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript
